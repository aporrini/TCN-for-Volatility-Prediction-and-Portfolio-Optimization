# TCN for Volatility Prediction and Portfolio Optimization

## Description

This project implements a *Temporal Convolutional Network* (TCN) for volatility prediction of financial instruments. These predictions are then used to power an algorithm aimed at optimizing the returns of the S\&P 500 by leveraging its Sector ETFs.

## System Requirements

* Python 3.8 or higher

### Python Packages:

* numpy
* pandas
* torch (PyTorch)
* matplotlib
* scikit-learn

Install all dependencies with:

```bash
pip install -r requirements.txt
```
